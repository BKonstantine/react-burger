# Stellar burgers

[Stellar burgers](https://react-burger-inky.vercel.app/) - это десктопная версия сайта бургерной. Сайт позволяет лично сконструировать бургер и офрмить заказ.

## Технологии
* TypeScript
* React
* Redux
* React Router
* React-dnd
* Webpack
* WebSocket

## Запуск проекта
Установить зависимости
```
npm install
```

Запустить локальный сервер
```
npm start
```
